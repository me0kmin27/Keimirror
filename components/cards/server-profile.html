Server profile

Mirror 1 and Mirror 2 are separate Gumi-based servers. Both mirrors use a CDN only for DDoS mitigation, with CDN caching disabled so files are served directly from the origin mirrors.

Mirror 1

CPUIntel Xeon E5-2650 v2 (2 sockets)
RAM128 GB
Storage10 TB
Network2 WAN links, 1 Gbps each / 2 Gbps total
LocationGumi, South Korea
CDNCDN-enabled for DDoS mitigation only; CDN caching is not used.
