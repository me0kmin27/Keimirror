Keimirror

Keiminem 미러 서비스

Keimirror는 웹과 rsync 접속 정보를 한 곳에서 확인할 수 있도록 구성한 미러 안내 허브입니다. 왼쪽에는 서비스 소개와 운영 안내를, 오른쪽에는 바로 사용할 수 있는 서버 접속 정보를 정리했습니다.

서버 소개

Keiminem 미러는 패키지, 릴리스, 오픈소스 프로젝트 파일 등을 빠르게 내려받을 수 있도록 운영되는 공개 미러 서비스입니다. 방문자는 웹 브라우저에서 디렉터리 목록을 확인하거나, 자동화된 동기화 작업에는 rsync 엔드포인트를 사용할 수 있습니다.

각 미러 서버는 동일한 목적의 파일 목록을 제공하며, 사용자는 네트워크 상황이나 접근 방식에 맞춰 기본 미러와 보조 미러 중 하나를 선택할 수 있습니다.

웹 접근

rsync 동기화

대량 파일 동기화나 자동화 스크립트에는 rsync 전용 주소를 사용하는 것을 권장합니다.
